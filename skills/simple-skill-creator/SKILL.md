---
name: simple-skill-creator
description: "Create or update a simple skill. Use when the user wants a lightweight skill with only the required `SKILL.md` format: YAML frontmatter with `name` and `description`, plus a one-paragraph body, without extra sections, resources, or files unless explicitly requested."
---

Create the skill as a simple `SKILL.md`. Use only YAML frontmatter with `name` and `description`, then write the body as one paragraph. Put trigger and when-to-use information in `description`. Keep the skill direct and minimal. Do not add extra sections, extra files, or resource folders unless the user explicitly asks for them.
